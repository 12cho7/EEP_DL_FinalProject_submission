# Checkpoints Directory

This directory is a placeholder for saved model weights.

## Model Location

The trained model (v3.0) should be placed here after downloading:

```
checkpoints/
└── models_v3/          # Extract the downloaded models_v3.zip here
    ├── adapter_config.json
    ├── adapter_model.bin
    ├── tokenizer_config.json
    ├── spm.model
    └── config.json
```

## Download Instructions

1. Download the model from the link provided in the main README.md
2. Extract `models_v3.zip` to this directory
3. The final structure should be: `checkpoints/models_v3/`

## Model Size

- Uncompressed: ~700MB
- Compressed (zip): ~200-300MB
